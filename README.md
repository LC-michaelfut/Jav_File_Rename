# Jav_File_Rename
Python desktop app that auto-renames censored JAV (番号) video files, with performers and title in Japanese, by scraping metadata from JAVLibrary &amp; AVMOO. Supports studio IDs like MIGD, DASD, PIYO etc. Features Cloudflare bypass, smart ID extraction, multi-part detection, and persistent rename logging. Uncensored studios not supported.
